# 🌍 Totoversum

Totoversum on Three.js:lla, React Three Fiberillä ja TypeScriptillä toteutettu rentouttava, kosminen pulmapeli. Rakenna omaa planeettaasi asettamalla biomeja, luomalla ketjureaktioita (merge) ja keräämällä pisteitä parhailla biosisynergioilla.

## 🎵 Äänet & Musiikki
Peli sisältää täysin integroidun äänimaailman:
- **Taustamusiikki**: Rentouttava, kosminen taustamusiikki loopilla.
- **Syntetisoidut ääniefektit**: Web Audio API:n avulla reaaliajassa luodut kosmiset äänet palikoiden asettamiselle, yhdistämiselle (merge), häviämiselle (game over) sekä nollaukselle.
- **Asetuspaneeli (⚙️)**: Voit säätää taustamusiikin ja ääniefektien voimakkuutta erikseen sivun ylälaidan asetusvalikosta.

## 🛠️ Kehitys ja käynnistys
Asenna riippuvuudet ja käynnistä paikallinen kehityspalvelin:
```bash
npm install
npm run dev
```

Rakenna optimoitu tuotantopaketti:
```bash
npm run build
```
